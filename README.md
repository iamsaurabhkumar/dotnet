## ğŸš€ Getting Started

### Prerequisites
- .NET SDK 6.0 or later installed
- Any code editor (Visual Studio, VS Code, Rider, etc.)

### Running the Project

1. **Navigate to the project directory:**
   ```bash
   cd CSharpLearning
   ```

2. **Run the project:**
   ```bash
   dotnet run
   ```

3. **Select from the menu:**
   - Choose a category (1-7) to explore
   - Option 7 runs all examples
   - Option 0 exits

### Building the Project

```bash
dotnet build
```
